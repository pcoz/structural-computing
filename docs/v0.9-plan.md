## 2. Deliverable detail

### Deliverable 1 — Full LT 1979 with explicit planar-dual

**File:** `structural-computing/structural_computing/decompose.py`.

The Lipton-Tarjan auto-separator cascade now has FIVE tiers:

1. **v0.4** — simple BFS-layer case.
2. **v0.5** — spanning-tree tree-edge backup.
3. **v0.6 D2** — level-based + articulation-augmentation backup.
4. **v0.8 D2** — fundamental-cycle backup (Jordan-curve implicit).
5. **v0.9 D1** (this release) — full LT 1979 planar-dual backup.

**Algorithm of the fifth tier:**

1. Trace faces from the rotation system using holant-tools'
   `genus_from_rotation_system`. Verify genus 0 (planar).
2. Build the edge-to-faces map: each undirected primal edge
   bounds exactly two faces in a 2-connected planar embedding.
3. Build the vertex-to-faces map.
4. Construct the primal BFS spanning tree T from the BFS levels
   already computed by the cascade dispatch.
5. The cotree primal edges form (their duals form) a spanning
   tree T* of the planar dual graph G*. (Classical fact:
   F - 1 = m - n + 1 in genus 0.)
6. For each cotree primal edge e (= dual edge e* in T*):
   a. Remove e* from T*; BFS the dual to identify F_inside
      (connected component containing the face e* borders on
      one side).
   b. F_outside = complement.
   c. For each primal vertex v: incident faces classification.
      All inside → v ∈ A. All outside → v ∈ B. Mixed → v ∈ S
      (on the fundamental cycle).
   d. Validate |S| ≤ 2·sqrt(2n), max(|A|, |B|) ≤ 2n/3.
   e. Score: smaller |S|, more balanced (A, B); track best.

**Cascade dispatch update.** `_lipton_tarjan_separator` now reads
`problem["rotation"]` when present and invokes the 5th tier after
the 4th tier fails. Without rotation, the cascade ends at the 4th
tier (same as v0.8 behaviour).

**Reuses holant-tools machinery:** `genus_from_rotation_system`,
face-cycles tracing, edge-to-faces map construction.
